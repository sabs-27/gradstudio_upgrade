Step 5: Cloudflare Workers

The request hits a Cloudflare Worker. This is your backend logic, but it's not a server in a basement. It's a function that runs on the Edge.

// worker.js
export default
{
fetch(request, env) {
return new Response("Hello from the Edge!");
}
}
This single file handles API endpoints like /api/courses and /api/likes.
